# Object Passing

## Question #1

What will be the output of this code?

```ruby
def change_name(name)
  name = "bob"
end

name = "jim"
change_name(name)
puts name
```


## Question #2

What will this code print?

```ruby
def add_to_array(arr)
  arr << "new item"
end

my_array = ["old item"]
add_to_array(my_array)
p my_array
```


## Question #3

What will be the value of `a` after this code runs?

```ruby
def modify_string(str)

a = "hello"
modify_string(a)
```
