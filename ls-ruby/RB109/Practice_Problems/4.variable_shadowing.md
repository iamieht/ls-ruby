# Variable Shadowing

## Question #1

What does the following code return? What does it output? Why? What concept does it demonstrate?

```ruby
animal = "dog"

loop do |_|  
  animal = "cat"  
  var = "ball"  
  break
end

puts animal
puts var
```

## Question #2

What does the following code return? What does it output? Why? What concept does it demonstrate?

```ruby

	puts a
end
puts a
puts b
```
