
# Short-Circuit evaluation

## Question #1

What will be the value of `result` after this code runs?

```ruby
a = nil
b = "Hello"
result = a || b
```


## Question #2

What will this code output?

```ruby
x = false
y = "World"
puts x && y
```


## Question #3

What will be the value of `z` after this code executes?

```ruby
z = true && false || true
```


## Question #4

What will this code print?

```ruby
def greet(name)
  puts "Hello, #{name}!"
end

name = nil
greet(name || "Guest")
```


## Question #5

What will be the output of this code?

```ruby
a = []
b = "Not empty"
puts a.empty? && b
puts a.empty? || b
```
