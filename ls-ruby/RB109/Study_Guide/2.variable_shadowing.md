# Variable Shadowing

* Prevents access and modification to the outer scope local variable.
* Avoid variable shadowing.

### Example

```ruby
n = 10

[1, 2, 3].each do |n|
  puts n
end
```

The `puts n` will use the block parameter `n` and disregard the outer scoped local variable.

```ruby
n = 10

1.times do |n|
  n = 11
end

puts n          # => 10
```
